# Papers

Five documents covering why 7fchain exists, how it works, and the principles it runs by. They're written to be readable independently but work best in this order.

## Suggested reading order

| # | Paper | What it covers | Length |
|---|-------|---------------|--------|
| 1 | [Mission & Design](7fchain-mission-and-design.md) | Why 7fchain exists - the problem with legacy payment corridors and the design choices that follow from it | Short |
| 2 | [Executive Summary](7fchain-executive-summary.md) | The whole project in a few pages - L1, L2, tokenomics, and team. Start here if you want the overview first | Short |
| 3 | [L1 White Paper](7fchain-l1-white-paper.md) | The technical design: post-quantum cryptography, proof-of-work, certificate system, block structure, consensus | Long |
| 4 | [7fCOIN Tokenomics](7fcoin-tokenomics.md) | Supply cap, block rewards, cooldown, tail emission, and 7fUSD | Medium |
| 5 | [Constitution](7fchain-constitution.md) | The ethos, the commitments we make to miners, and the rules we hold ourselves to | Short |

Each paper is available as Markdown (readable here on GitHub and by AI agents) and as a PDF - both formats are in this directory.

## If your time is short

- **5 minutes:** read the [Executive Summary](7fchain-executive-summary.md).
- **30 minutes:** add the [Mission & Design](7fchain-mission-and-design.md) and [Tokenomics](7fcoin-tokenomics.md).
- **Deep dive:** all five, in order.
